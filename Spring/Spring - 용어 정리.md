## JAXB

```
1. 기능 : JAXB, Java Architecture for XML Binding : 자바 클래스를 XML로 표현하는 자바 API. 자바 객체를 XML로 직렬화하거나 반대로 XML에서 자바 객체로 역직렬화하는 기능을 갖는다.

2. 특징
	1) JDK 1.8에는 포함되어 있었지만 JDK 10 부터는 포함되지 않았다

```
