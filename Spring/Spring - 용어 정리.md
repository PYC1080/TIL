```
	2) 람다식 선언
		(1) 기본 구조 : (타입 매개변수,...) -> {실행문;...};
		(2) 생략 기법
			a) 람다식 매개인자의 자료형은 생략이 가능하다
			b) 람다식의 매개인자가 한개인 경우 매개인자를 감싸는 소괄호를 생략할 수 있다
			c) 람다식의 함수몸체에 실행문이 한개인 경우 함수의 몸체를 감싸는 중괄호를 생략할 수 있다
			d) 람다식의 함수몸체에 실행문이 한개이고, 그 실행문이 return문인 경우 함수의 몸체를 감싸는 중괄호와 return을 생략할 수 있다.
```





## JAXB

```
1. 기능 : JAXB, Java Architecture for XML Binding : 자바 클래스를 XML로 표현하는 자바 API. 자바 객체를 XML로 직렬화하거나 반대로 XML에서 자바 객체로 역직렬화하는 기능을 갖는다.

2. 특징
	1) JDK 1.8에는 포함되어 있었지만 JDK 10 부터는 포함되지 않았다

```
