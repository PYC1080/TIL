# JavaScript - tutorial

## 1. Intro

> JavaScript로 할 수 있는 일들

```
1. Javascript can change HTML content 
2. Javascript can change HTML attribute values
3. Javascript can change HTML Styles(css)
4. Javascript can change HTML Elements
```

## 2. Where to

> JavaScript 코드  위치

```
1. HTML 에선 javascript code를 <script></script>사이에 작성한다
2. Javacript 함수와 이벤트는 호출시 실행된다
3. <script>태그는 <head>와 <body> 태그에 들어갈 수 있으며 갯수 제한은 없다.
4. 외부 자바스크립트를 만들고 불러와 사용할 수 있다.
```

## 3. Output

> JavaScript 코드 산출물

```
1. innerHTML : HTML elements 표현
2. document.write()
	1) HTML output 표현시 사용
	2) HTML document가 로드되면 기존 모든 HTML은 삭제된다
3. window.alert() : alert box
4. console.log() : browser conosle
```

## 4. Statements

```
1. JavaScript statements are composed of
	1) values
	2) operators
	3) expressions
	4) keywords
	5) comments

2. Semicolons(;) : JS statements를 구분하는 기능

3. white space : JS는 공백을 무시한다

4. Code blocks : {}을 사용해 코드를 그룹화 할 수 있다.

5. keywords
	1) break : switch나 loop를 중단시킴
	2) continue : loop 반복을 중단하고 처음 시작점으로 돌아감
	3) debugger : JS 실행을 멈추고 debugging function을 호출한다
	4) do..whie : 조건이 true인 경우 block statments를 반복해서 실행한다.
	5) for : 조건이 true인 경우 지정된 block statments를 실행한다
	6) function : 함수 선언
	7) if...else : 조건에 따라 지정된 block statements를 실행한다
	8) return : 함수 종료함
	9) switch : 서로다른 cases에 따라 지정된 block statements를 실행한다
	10) try...catch : block statements에 대한 오류 처리 구현
	11) var : 변수 선언
```

## 5. Syntax

```
1. literals vs variables
	1) literals : fixed value
	2) variables : variable value
	
2. Comments : // or /**/

3. Identifiers
	1) 변수의 첫 글자는 반드시 문자열/_/$ 로 시작한다
	2) 특수문자는 _/$만 사용이 가능하다
	3) 대소문자를 구분한다
	
4. Camel case
	1) Underscore
	2) Upper camel case(Pascal Case) : 두 단어 이상의 조합으로 변수를 선언하는 경우 각 단어의 첫글자를 대문자로 작성(e.g.,FirstName)
	3) Lower camel case : 두 단어 이상의 조합으로 변수를 선언하는 경우 첫 단어의 첫글자는 소문자로 첫 단어 이후의 첫글자는 대문자로 작성(e.g.,firstName)
	
5. Character set : Unicode
```

## 6. Operators

```
1. Arithmetic operators
 	1) addition : +
    2) subtraction : -
    3) multiplication : *
    4) exponentiation : **
    5) division : /
    6) division remainder : %
    7) increment : ++
    8) decrement : --
    
2. Assignment operators
	1) x=y : x=y
	2) x+=y : x=x+y
	3) x-=y : x=x-y
	4) x*=y : x=x*y
	5) x/=y : x=x/y
	6) x%=y : x=x%y
	7) x**=y : x=x**y
	
3. Comparison operators 
	1) equal to : ==
	2) equal value and equal type : ===
	3) not equal : !=
	4) not equal and not equal type : !==
	5) greater than : >
	6) less than : <
	7) greater than or equal to : >=
	8) less than or equal to : <=
	9) ternary operator(삼항 연산자) : condition ? true : false
	
4. Logical operators
	1) logical and : &&
	2) logical or : ||
	3) logical not : !
	
5. Type operators 
	1) retuns the type of a variable : typeof
	2) return true if an object is an instance of an object type 
```
